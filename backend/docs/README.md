# 文档目录

本目录包含项目的所有文档，分为两个主要类别：

## 目录结构

### usage/ - 使用文档
各个模块和服务的使用说明文档，帮助开发者了解如何使用各个组件。

- `AUDIO_EXTRACTOR_USAGE.md` - 音频提取服务使用指南
- `CELERY_TASKS_USAGE.md` - Celery 异步任务使用指南
- `EMBY_CONNECTOR_USAGE.md` - Emby 连接器使用指南
- `SUBTITLE_GENERATOR_USAGE.md` - 字幕生成服务使用指南
- `TASK_MANAGER_USAGE.md` - 任务管理服务使用指南
- `TRANSLATION_SERVICE_USAGE.md` - 翻译服务使用指南

### implementation/ - 实现文档
记录各个任务的实现过程、验证结果和技术细节。

- `API_IMPLEMENTATION.md` - API 实现文档
- `TASK_2_IMPLEMENTATION.md` - 任务2实现文档
- `TASK_3.1_VERIFICATION.md` - 任务3.1验证文档
- `TASK_4.1_IMPLEMENTATION.md` - 任务4.1实现文档
- `TASK_7.1_IMPLEMENTATION.md` - 任务7.1实现文档
- `TASK_9_IMPLEMENTATION.md` - 任务9实现文档
- `TASK_11_IMPLEMENTATION.md` - 任务11实现文档
- `TASK_11_SUMMARY.md` - 任务11总结文档
- `TASK_12_VERIFICATION.md` - 任务12验证文档（后端核心功能检查点）

## 文档规范

### 使用文档格式
使用文档应包含：
1. 模块概述
2. 主要功能
3. 使用示例
4. API 参考
5. 常见问题

### 实现文档格式
实现文档应包含：
1. 任务目标
2. 实现方案
3. 技术细节
4. 测试结果
5. 遇到的问题和解决方案
